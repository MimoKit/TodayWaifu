> 插件交流请加群：[798949533](https://qm.qq.com/q/ejzCUfJ5le)

## 使用说明

具体功能和命令请给 bot 发送：

```text
今日老婆帮助
```

bot 会返回插件帮助图，按帮助图里的说明使用即可。
